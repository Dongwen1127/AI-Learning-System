# AI助学系统
这是一个基于大语言模型的个性化学习辅助系统，它通过一个轻量化的Python Web应用实现，将智能AI能力无缝嵌入“课前预习、课中学习、课后复习”的全流程。系统后端采用Python处理核心逻辑与API调用，前端通过HTML/CSS/JavaScript构建交互界面，并利用大语言模型生成个性化内容。其核心用途是为教师和学生提供一个“AI助学”伙伴，在课前生成分层预习材料，课中提供实时编程调试与概念讲解，课后基于学情分析推送个性化巩固练习，从而将教师从重复性工作中解放出来，专注于启发与引导，并实现规模化因材施教。
